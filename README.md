AI-PERSONALITY-STUDY-ANALYZER

##📌Problem Statement

Many Students struggle to understand their personality and study stlye, leading to low productivity and ineffective learing methoods.


##📔Solutions

This project analyzes a student's personality using a questionnaire and provides personalized study recommendations.


##⚙️Features

-Personality predictions (MBTI - based)
-Study style suggestions
-Strengths and weaknesses analysis
-Unsuitable study environment detection


##🛠️Technology used

-Python
-Rule based AI logic


##▶️How to Run

1.Download the project
2.Open terminal
3.Run:
 python main.py
